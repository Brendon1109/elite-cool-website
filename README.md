### Step 1: Create Repository

2. Click "New Repository"
3. Name it: `elite-cool-website`
4. Make it public
5. Initialize with README (optional)

### Step 2: Upload Files
1. Clone the repository or upload files directly
2. Add all project files to the repository
3. Commit and push changes

### Step 3: Enable GitHub Pages
1. Go to repository Settings
2. Scroll to "Pages" section
3. Under "Source", select "GitHub Actions"
4. The site will be available at: `https://brendon1109.github.io/elite-cool-website/`

### Step 4: Add Images
1. Add your business images to the `assets/` folder
2. Replace placeholder images with actual photos
3. Commit and push changes
