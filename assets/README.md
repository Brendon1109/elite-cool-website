# ELITE COOL Website Assets

## Images Needed

Please add the following images to this folder to complete the website:

### Required Images:
1. **hero-image.jpg** (1200x800px) - Main hero section image showing refrigeration/AC equipment or technician at work
2. **about-image.jpg** (800x600px) - Image of ELITE COOL team or equipment for the About section
3. **favicon.ico** (32x32px) - Website icon for browser tabs

### Image Guidelines:
- Use high-quality images (minimum 1080p for hero image)
- Optimize images for web (compress to reasonable file sizes)
- Use professional photos that represent your business
- Consider the fresh & eco-friendly color palette (blues and greens)

### Recommended Image Sources:
- Professional photos of your actual work/team
- Stock photos from Unsplash, Pexels, or similar free resources
- Custom photography showcasing your refrigeration and AC services

### Image Optimization:
- Hero image: Max 500KB
- About image: Max 300KB
- Favicon: Max 10KB

### Fallback:
The website will still work without these images, but adding them will significantly improve the visual appeal and professionalism of the site.
